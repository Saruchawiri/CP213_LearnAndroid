# Decidr - UI Wireframes (Low-Fidelity)

เพื่อประกอบการส่งงานหรือใส่ในสไลด์พรีเซนต์ นี่คือการออกแบบโครงร่างหน้าจอ (Wireframes) ในรูปแบบ Text-based Blueprint เชิงเทคนิค ซึ่งสามารถคัดลอกใส่กล่องข้อความ (Text Box) แบบฟอนต์ Monospace (เช่น Courier New, Consolas) ลงในรายงานได้เลยครับ

## 1. Main Screen (หน้าหลักสำหรับการป้อนข้อมูล)
```text
+-----------------------------------------+
|  10:00                           LTE [] |
+-----------------------------------------+
|                                         |
|              [🐶 Jelly]                 |
|     "Hi! What are we deciding today?"   |
|                                         |
|  [ Your Question                      ] |
|  [ e.g., Where should we eat?         ] |
|                                         |
|  Options:                               |
|  [ Option 1: Sushi                  ] ✕ |
|  [ Option 2: Burger                 ] ✕ |
|  + Add Option                           |
|                                         |
|  [ 🟢 ] Advanced Mode (Factors)         |
|  -------------------------------------  |
|  [ Factor: Price                    ] ✕ |
|  Weight: -----O-----------------        |
|                                         |
|  [ Factor: Taste                    ] ✕ |
|  Weight: -------------O---------        |
|  + Add Factor                           |
|                                         |
|                                         |
|     ===============================     |
|     |       DECIDE FOR ME!        |     |
|     ===============================     |
|                                         |
|     [ View History ]                    |
+-----------------------------------------+
```

## 3. Chat Screen (หน้าสำหรับสนทนาต่อยอด)

Project Proposal

Mobile Application: Decidr

⸻

Step 1: Project Overview

1. ชื่อโครงการ

Decidr
แอปพลิเคชันช่วยตัดสินใจด้วย AI และการวิเคราะห์เชิงเหตุผล

⸻

2. ที่มาและความสำคัญของโครงการ

ในชีวิตประจำวัน ผู้คนมักต้องเผชิญกับการตัดสินใจในหลายรูปแบบ เช่น
	•	ควรซื้อสิ่งนี้หรือไม่
	•	ควรเลือกทำงานชิ้นไหนก่อน
	•	ควรเลือกตัวเลือกใดระหว่างหลายทางเลือก

แม้บางการตัดสินใจจะดูไม่ซับซ้อน แต่เมื่อมีหลายปัจจัยเข้ามาเกี่ยวข้อง เช่น ราคา เวลา ความคุ้มค่า หรือความรู้สึกส่วนตัว มักทำให้ผู้ใช้เกิดความลังเลและใช้เวลานานในการตัดสินใจ

แอป Decidr จึงถูกพัฒนาขึ้นเพื่อช่วยให้ผู้ใช้สามารถตัดสินใจได้อย่างมีเหตุผล โดยใช้แนวคิด AI-assisted decision making ร่วมกับ การคำนวณแบบถ่วงน้ำหนัก (Weighted Scoring System)

ระบบจะช่วย:
	•	วิเคราะห์ข้อดี–ข้อเสียของแต่ละตัวเลือก
	•	คำนวณคะแนนตามปัจจัยที่ผู้ใช้กำหนด
	•	แนะนำตัวเลือกที่เหมาะสม พร้อมเหตุผลประกอบ

นอกจากนี้ แอปยังมีฟีเจอร์
	•	Scenario Simulation (จำลองผลลัพธ์ในอนาคต)
	•	AI Chat Assistant (ผู้ช่วยถาม–ตอบ)

ทำให้ Decidr เป็นมากกว่าแอปทั่วไป แต่เป็น Intelligent Decision Support System ที่ช่วยให้การตัดสินใจมีประสิทธิภาพและแม่นยำมากขึ้น

⸻

3. วัตถุประสงค์
	•	เพื่อพัฒนา Mobile Application บน Android สำหรับช่วยตัดสินใจด้วย AI
	•	เพื่อฝึกการพัฒนาแอปด้วย Kotlin และ Jetpack Compose
	•	เพื่อเรียนรู้และประยุกต์ใช้ MVVM Architecture
	•	เพื่อออกแบบระบบ Decision Engine ด้วย Weighted Scoring
	•	เพื่อเชื่อมต่อ AI API สำหรับวิเคราะห์และให้คำแนะนำ
	•	เพื่อพัฒนา Explainable AI ที่สามารถอธิบายเหตุผลได้

⸻

4. กลุ่มเป้าหมาย
	•	นักศึกษา
	•	คนวัยทำงาน (อายุ 18–35 ปี)
	•	ผู้ที่ต้องตัดสินใจบ่อย
	•	ผู้ที่ต้องการเครื่องมือช่วยคิดอย่างมีเหตุผล

3.3 System Flow
	1.	ผู้ใช้ใส่คำถาม
	2.	เพิ่มตัวเลือก (≥ 2)
	3.	เพิ่มปัจจัย + weight
	4.	กด Analyze
	5.	ระบบคำนวณคะแนน
	6.	AI วิเคราะห์
	7.	แสดงผลลัพธ์
	8.	บันทึก History
	9.	ผู้ใช้ถามต่อได้

Project Proposal (Draft)
Mobile Application: Whatever in BKK
1. Project Overview
Project Name: Whatever in BKK
Project Type: Android Mobile Application
Platform: Android
Development Tools: Android Studio, Kotlin, Jetpack Compose

1.1 Background and Rationale
ในชีวิตประจำวัน ผู้คนมักประสบปัญหาในการตัดสินใจเรื่องเล็ก ๆ เช่น
จะรับประทานอาหารอะไร
จะไปสถานที่ใด
จะทำงานชิ้นใดก่อน

การลังเลหรือถกเถียงกันในกลุ่มเพื่อนอาจทำให้เสียเวลาโดยไม่จำเป็น แอปพลิเคชัน Whatever in BKK ถูกออกแบบมาเพื่อช่วยลดระยะเวลาในการตัดสินใจ โดยใช้ระบบสุ่ม (Random Decision) เพื่อเลือกตัวเลือกให้ผู้ใช้โดยอัตโนมัติ

2. Objectives
เพื่อพัฒนาแอปพลิเคชันช่วยตัดสินใจในชีวิตประจำวัน
เพื่อลดระยะเวลาในการลังเลและเพิ่มความสะดวกในการเลือก
เพื่อฝึกทักษะการพัฒนา Mobile Application ด้วย Kotlin และ Jetpack Compose
เพื่อเรียนรู้การจัดการข้อมูลภายในเครื่องด้วยฐานข้อมูล Room

3. Target Users
นักศึกษา
คนวัยทำงานช่วงอายุ 18–30 ปี
ผู้ที่ตัดสินใจได้ยาก
กลุ่มเพื่อนที่ต้องการวิธีตัดสินใจที่รวดเร็วและสนุกสนาน

4. Scope of Work (MVP Features)
4.1 Add Options
ผู้ใช้สามารถ:
เพิ่มตัวเลือกด้วยตนเอง
ลบตัวเลือกได้
กำหนดจำนวนตัวเลือกขั้นต่ำ 2 รายการ

4.2 Random Decision
มีปุ่ม “Decide” เพื่อสุ่มเลือก 1 ตัวเลือก
แสดงผลลัพธ์บนหน้าจออย่างชัดเจน
เพิ่ม animation เบื้องต้นเพื่อสร้างความน่าสนใจ

4.3 History
บันทึกผลลัพธ์การสุ่มล่าสุด
แสดงรายการย้อนหลัง
สามารถล้างประวัติได้

4.4 Offline Usage
ฟีเจอร์หลักทั้งหมดสามารถใช้งานได้โดยไม่ต้องเชื่อมต่ออินเทอร์เน็ต

5. System Architecture and Technology
5.1 Development Platform
Android Studio
Kotlin Programming Language

5.2 Database
Room Database (SQLite)
ใช้สำหรับเก็บข้อมูล History ภายในเครื่อง

5.3 Application Structure
Home Screen: เพิ่มและจัดการตัวเลือก
Result Screen: แสดงผลลัพธ์การสุ่ม
History Screen: แสดงผลย้อนหลัง

6. Expected Outcomes
ได้แอปพลิเคชัน Android ที่สามารถใช้งานได้จริง
ผู้ใช้สามารถตัดสินใจเรื่องเล็ก ๆ ได้รวดเร็วขึ้น
โครงสร้างโปรแกรมมีความชัดเจน และสามารถต่อยอดในอนาคตได้

7. Future Improvements (Beyond MVP)
เพิ่มธีมและการปรับแต่ง UI
เพิ่มชุดตัวเลือกสำเร็จรูป เช่น ร้านอาหารในกรุงเทพ
เพิ่มฟีเจอร์แชร์ผลลัพธ์ไปยัง Social Media
พัฒนาโหมด Multiplayer
